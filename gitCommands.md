# How to make a commit

There is a multi stage process here.

```
    git add .
    (Do ^ in the parent folder of the whole project)

    git commit -m "The message you want to do"
    (EG added health the enemy)
    
    git push
    (if it complains talk to justace.)
    (If you are in a branch make sure that you are in the master branch). 

```

# How to Make a branch

Open the terminal in the game folder.

In the terminal you want to write this command

```
    git checkout -b <new-branch-name>
```

# How to Check what branch you are currently on

Use this command to check what branch you are currently on

```
    git branch
```

it will list all the branchs that we have and have a little star next to the one that you are on for instance:

```
    master
    *test
    feature1
```

# How to Pull up to date files

Once we have done a pull request and you want the update to date files, just run this in the terminal

```
    git pull
```
